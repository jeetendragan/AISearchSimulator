Help Dialog

✓ Create nodes by double clicking or by clicking the scatter_plot button.
✓ Create edge by selecting the call_made button, and then selecting the nodes. To unselect the start node, press ESC or click in empty space.
✓ Define a start node and a goal node by selecting a node and choosing the option on the right.
✓ Select an algorithm and start the simulator.
✓ Zoom in by clicking on the zoom_in button or scroll down.
✓ Zoom out by clicking on the zoom_out button or scroll up.
✓ Go into the pan state by clicking pan_tool button and then drag the canvas OR Drag the canvas by pressing ALT key.
